# dotfiles

### MacOS Quickstart

```sh
export RCM_TAG=macos
export DOTFILES="$HOME/.dotfiles"

git clone git@github.com:awseward/dotfiles.git "$DOTFILES"
"$DOTFILES"/tag-$RCM_TAG/quickstart.sh
```
